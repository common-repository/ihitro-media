Ihitro Media klant informatie

== Description ==
Plugin voor klanten van Ihitro. Login pagina word aangepast en men kan ook informatie over Ihitro Media vinden.

Ordered list:

* Ihitro Media login scherm
* Bedrijf gegevens


== Installation ==
1. Installeer de map in "/wp-content/plugins/".
1. Activeer de plugin bij "Plugins" in je WordPress installatie.
1. Je kan gebruik maken van de plugin.
